E-commerce

This project was completed as part of a training programme with Wiley Edge, and as the final project completed on the training programme it was intended to showcase the variety of tools and technologies that had been studied. The front end of the website was built using HTML, CSS and JQuery, while Thymeleaf was used to integrate the back end functionality, which was coded in Java with Spring boot and Spring MVC. A MySQL database was also created to store information about products and customers' orders. The website was designed with a retro and simplistic feel, and included a landing page displaying recent additions, as well as options to search records and filter by artist and genre.
